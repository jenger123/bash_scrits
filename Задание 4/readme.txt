1. файлы ex4.sh text1.txt text2.txt закинуть в один каталог в linux. Например в домашнюю директорию пользователя /home/<username>/
2. открыть терминал и переместиться в домашний каталог: cd /home/<username>
3. дать скрипту права на исполнение: sudo chmod 777 ex4.sh
4.запустить скрипт на исполнение: bash ex4.sh <размер шинглов> <файл1> <файл2> <файл-словарь 1> <файл-словарь 2> <файл отчет>
Пример: bash ex4.sh 3 text1.txt text2.txt dict1.csv dict2.csv report.txt
После того как скрипт отработал, покажите результаты преподавателю:
cat dict1.csv
cat dict2.csv
cat report.txt

В коде скрипта есть подробные комментарии. Рекомендую показывать на моих файлах text1.txt text2.txt, но при желании можете написать свои файлы
