Как работает эта штука:

1.Скачать Virtual box: https://www.virtualbox.org/wiki/Downloads
2. Скачать Linux(например Ubuntu: https://ubuntu.com/download/desktop)
3. Установить убунту
4. перенести файл скрипта и словаря ex1.sh и excludes.txt соответственно в убунту в домашнюю директорию (по умолчанию /home/{username}
5. Найти или создать файл с русским текстом (в архиве есть файл поменьше kon.txt и побольше tihiy_don.txt)
6.Откройте терминал. Для того, чтобы файл скрипта смог выполниться требуется дать права доступа. наберите команду: sudo chmod +rwx ex1.sh или sudo chmod 777 ex1.sh
7. Чтобы запустить скрипт наберите команду bash ex1.sh <размер шингла> <имя файла, который нужно разбить на шинглы> <имя рабочего файла> <имя файла отчета> <имя файла-словаря исключений>
Пример: bash ex1.sh 4 kon.txt workfile.txt report.txt exclusion.txt
	|     (обязательно) |			      |   обязательно |

Чтобы продемонстрировать результат работы нужно показать рабочий файл: cat workfile.txt
и файл отчет: cat report.txt
при  желании можно протестировать на большом файле (Тихий дон 1 и 2 том файл tihiy_don.txt)
