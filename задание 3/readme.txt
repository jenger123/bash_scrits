*запуск сценария
помещаем скрипт в любую папку (лучше всего /home/<username>)
открываем терминал, пишем: cd ~   (~ там где буква "ё" - переход в домашнюю директорию)
пишем : chmod +x ex3.sh или chmod 777 ex3.sh
запуск самого скрипта производится командой:
bash ex3.sh <директория в которой нужно начать поиск> <имя файла-отчета> <пороговое значение энтропии>

Пример:
bash ex3.sh /home/leonid/test/ /home/leonid/report.csv 2

значение энтропии положительное, у архивов и криптоконтейнеров этот параметр будет очень большим.

чтобы продемонстрировать результат работы напишите cat /home/leonid/report.csv
